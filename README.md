## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/actuing/ST-T1024-APPLICATIONS.git
   cd ST-T1024-APPLICATIONS
   ```

2. **Read any application:**
   ```bash
   cat 01-EDGE-AI-GENERALITIES.md
   ```

3. **Deploy the application using the ST-T1024 emulator:**
   ```bash
   git clone https://github.com/actuing/ST-T1024-STANDARD-FOR-EMBEDDED-LLM.git
   cd ST-T1024-STANDARD-FOR-EMBEDDED-LLM
   python3 IMPROVED-EMULATOR-of-ST-T1024/improved_st_t1024_v2.1.py
   ```

4. **Build your own application:**
   - Use the template in `/templates/application-template.md`
   - Submit via pull request
   - Or publish independently — the license allows it
